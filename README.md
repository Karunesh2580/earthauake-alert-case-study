    <h1>Earthquake Alert Case Study</h1>

    <p>This project presents a data-driven analysis of earthquake alerts using Python visualizations. It explores the relationship between alert levels, magnitude, depth, and public impact metrics like CDI and significance score.</p>

    <h2>Project Files</h2>
    <ul>
      <li><strong>Earthquake_Visualization.ipynb</strong> – Jupyter notebook with all visualizations</li>
      <li><strong>Earthquake_Case_Study.pdf</strong> – Final report with insights and conclusions</li>
      <li><strong>earthquake_alert_balanced_dataset.csv</strong> – Source dataset</li>
    </ul>

    <h2>Technologies Used</h2>
    <ul>
      <li>Python (Pandas, Seaborn, Matplotlib)</li>
      <li>Jupyter Notebook</li>
      <li>Hive (for initial query and export)</li>
      <li>GitHub (for version control and hosting)</li>
    </ul>

    <h2>Key Visualizations</h2>
    <ul>
      <li>Distribution of earthquakes by alert level</li>
      <li>Magnitude variation across alert levels</li>
      <li>Depth vs Magnitude scatter plot</li>
      <li>Average significance score by alert level</li>
      <li>Top 10 earthquakes by CDI score</li>
    </ul>

    <h2>Conclusion</h2>
    <p>The analysis reveals that higher alert levels are associated with stronger and more impactful earthquakes. Public intensity metrics like CDI offer valuable insights beyond technical magnitude, aiding in disaster preparedness and response planning.</p>
